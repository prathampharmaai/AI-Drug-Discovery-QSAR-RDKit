# AI Drug Discovery using QSAR & RDKit
### Python | RDKit | scikit-learn | Pandas | Jupyter Notebook

## 📌 About This Project
A self-initiated computational drug discovery project 
combining pharmaceutical science knowledge with Python 
programming. Focused on building QSAR models and 
molecular analysis pipelines using RDKit.

## 🔬 What This Repository Contains

| Notebook | Description |
|---|---|
| QSAR.ipynb | QSAR model building for drug activity prediction |
| RD_kit_learning.ipynb | Molecular descriptor calculation using RDKit |
| Clean_TimololAdded_.ipynb | Drug dataset cleaning - Timolol analysis |
| full_pipeline_1.ipynb | End-to-end drug discovery pipeline |
| NumPy_stats.ipynb | Statistical analysis for pharma data |
| pandas_day_1.ipynb | Data manipulation for molecular datasets |
| plote.ipynb | Data visualization for drug properties |

## 🧪 Drug Used for Case Study
**Timolol** — Beta-blocker used in glaucoma treatment
